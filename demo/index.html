Quick Start

Add this script to your website's <head> section:

<script src="dist/cmp.min.js" data-site-id="demo-site-123"></script>

Manual Script Blocking

Mark scripts with data-category to control when they execute:

<!-- Analytics script (executes only with analytics consent) --> <script type="text/plain" data-category="analytics"> console.log('Analytics initialized'); </script> <!-- Marketing script (executes only with marketing consent) --> <script type="text/plain" data-category="marketing"> console.log('Marketing pixel loaded'); </script>
